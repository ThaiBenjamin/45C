## 💡 What It Taught Me

This course gave me my first real experience with manual memory management and pointer arithmetic — concepts that higher-level languages like Python abstract away entirely. Implementing a singly-linked queue from scratch in HW03 was the moment it clicked for me how pointers work. HW05's diamond inheritance problem (the "diamond problem" in OOP) gave me hands-on exposure to one of C++'s most nuanced features and why `virtual` inheritance exists. By the end, writing generic code with templates felt like a natural extension of what I already understood.
